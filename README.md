## Overview

**Pipedrive CRM** is a sales-focused CRM platform designed to help teams manage deals, contacts, activities, and sales pipelines in a simple visual workspace.

It is especially useful for small and mid-size sales teams that want a clean CRM focused on closing deals.

---

## Key Features

- Visual sales pipeline
- Deal tracking
- Contact management
- Activity reminders
- Email integration
- Sales automation
- Reports and insights
- Mobile CRM access
